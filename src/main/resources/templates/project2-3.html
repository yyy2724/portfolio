Project2-3

프리랜서 / 업체 연결 에이전시

소개

프리랜서와 회사를 연결하는 중개 에이전시 프로젝트 입니다.

프로젝트 기능

계약서 작성, 결제 처리 및 금융 관련
서비스를 제공

기간 / 인원

2023.09~2023.10 (약 1달)
총 6인

개인기여

로그인, 이메일 인증, 비밀번호 재설정,
권한별 LIST, 로그인&회원가입 디자인


이메일 인증

JavaMailHandler 클래스는 이메일 주로 이메일 발송, 수신, 첨부 파일 처리,
이메일 템플릿을 사용한 메시지 작성 등과 관련된 작업들을 처리 할 수 있습니다.
이 프로젝트에서는 인증번호 발송에 사용하였습니다.
글 작성자와 현재 접속자를 비교하여 알맞은 권한 설정을 하였습니다.


우편번호 API

카카오에서 제공하는 우편번호 API를 통하여 더 편리하게
주소정보를 입력하도록 구현하였습니다.


동작제어

JavaScript의 기능을 사용하여 이메일 미인증, 중복 이메일,
잘못된 인증번호 입력 등의 회원가입 조건 미충족시 회원가입이
불가능 하도록 구현하였습니다.



인증

회원가입과 같이 이메일을 인증하여 비밀번호 재설정 페이지에 진입합니다.


비밀번호 변경

새로운 비밀번호를 입력하고 비밀번호를 재입력하여 확인합니다.
두 비밀번호가 같지 않다면, 재설정이 되지 않습니다.
이러한 동작들은 JavaScript를 사용하여 제어하였습니다.



Spring Security

spring 기반의 애플리케이션에서 보안을 처리하기 위한 프레임워크입니다.
주로 웹 애플리케이션에서 사용되며, 사용자 인증(Authentication) 및 권한 부여(Authorization)와 관련된 다양한 보안 기능을 제공합니다.
이 프로젝트에는 페이지 권한 설정과 로그인에 사용되었습니다..


OAuth2.0

웹 및 애플리케이션에서 서비스의 허가된 접근을 위한 업계 표준 인증 프로토콜입니다.
OAuth 2.0은 사용자가 자신의 데이터에 대한 다른 서비스나 애플리케이션에 대한 제한된 액세스 권한을 부여하는 데 사용됩니다.
OAuth2.0 으로 모든 정보를 가져올 수 없으니 추가 정보를 입력해야 합니다.



Weather API

날씨 정보를 얻기 위해 사용되는 웹 API입니다.
이 API를 통해 날씨 관련 데이터를 요청하고 응답받을 수 있습니다.
다양한 날씨 API가 존재하며, 각각의 API는 특정 서비스 또는 날씨 데이터 제공 업체에서 제공하는 정보에 기반하고 있습니다.
