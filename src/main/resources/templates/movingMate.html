이삿짐 센터 프로젝트

소개

고객이 이사 출발 위치와 도착 위치, 이사시간 등을 정하면 회원인 이사 업체에서 금액 견적을 내줍니다. 고객은 견적을 확인후 결제를 할 수 있습니다.

프로젝트 기능

기업 회원가입 및 로그인, 캘린더, 주문,
가격 책정, 결제, Git Actions 를 통한 배포

기간 / 인원

2023.12~2024.01 (약 30일)
총 2 -> 1인 (초반부에 2인 팀 해체)

개인기여

프로젝트의 모든 기능


메인페이지 / 이사서비스 주문 / 주문내역 검색

FullCalendar를 사용하여 웹 애플리케이션을 동적이고 시각적으로 매력적인 이벤트 캘린더를 구현하였습니다.
모달 창을 이용하여 팝업 창을 구현하였습니다. 로그인, 주문찾기 기능에서 사용되었습니다.



기업 금액설정

기업이 견적을 확인 후 금액을 설정합니다.
이 과정은 RestFul 하게 디자인하여 새로고침 없이 값이 변경되도록 하였습니다.



결제

아임포트API를 이용하여 결제 시스템을 구현하였습니다.
업체에서 제시한 금액이 수락할 시 결제가 진행됩니다..
