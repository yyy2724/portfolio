Project1

마켓컬리 홈쇼핑

소개

‘마켓컬리’를 벤치마킹 한 의류판매 소핑몰 구현하였습니다.
webSocket을 통한 챗봇 상담 기능을 추가하였습니다.

프로젝트 기능

회원가입 및 소셜 로그인
주문, 장바구니, 공지사항 CRUD 등

기간 / 인원

2023.08~2023.09 (약 1달)
총 5 -> 3인

개인기여

쇼핑몰 공지사항 / 댓글 CRUD


프로젝트에서는 공지사항 게시글 / 댓글 CURD와
글 작성자와 현재 접속자를 비교하여 알맞은 권한 설정을 하였습니다.
웹 소켓을 이용한 챗봇 기능입니다.

WebSocket은 클라이언트와 서버(브라우저 서버)를 연결하고 실시간으로
통신이 가능하도록 하는 기술입니다.
HTTP에서 발생하는 것처럼 별도의 요청을 하지 않고도 데이터를 수신가능
데이터가 서버에서 클라이언트로 ‘푸시‘ 되므로, 실시간 업데이트에 효과적입니다.
